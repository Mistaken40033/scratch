Html

HTML short for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. It describes the structure of a Web page.

CSS

CSS,short for Cascading Style Sheets. , is a language used to control the appearance and layout of documents written in markup languages like HTML. It enhances HTML by defining how elements should be styled and formatted. Typically used alongside HTML, CSS enables developers to customize the design and appearance of web pages and user interfaces.

JavaScript

JavaScript is a scripting language utilized to enhance web pages by adding interactive elements and dynamic functionality.
